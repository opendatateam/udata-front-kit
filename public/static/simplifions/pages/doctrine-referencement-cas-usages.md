<h2 id="referencement-cas-usage" class="fr-h2 fr-my-0w" style="color: black; background-color: rgb(167, 212, 205); padding: 2px 4px; display: inline-block;">Règles de référencement d'un cas d'usage</h2>



<p class="fr-text--lead">Lorsque l'équipe de Simplifions.data.gouv.fr identifie un cas d'usage (par exemple : une démarche utilisateur ou un cadre d'utilisation qui requierent de la donnée), susceptible d'être simplifié via l'utilisation de données, voici les questions qu'elle se pose pour identifier si ce cas d'usage peut être référencé sur Simplifions.data.gouv.fr. </p>


<div class="fr-col-12 fr-col-lg-8 fr-ml-4w">
    <ol class="fr-text--lg">
        <li>
            <p class="fr-pt-4w"><span  class="fr-text--lg fr-text--bold">Ce cas d'usage concerne-t'il les acteurs publics* ? <br></span><i>* Administrations, collectivités, opérateurs, entreprise chargée d’une mission de service public, etc.</i></p>
            <ul>
                <li class="fr-text--lg"><span style="background-color: #bfd9e3ff; padding: 2px; border-radius: 4px;"><b>NON</b></span> Ce cas d'usage concerne des acteurs privés, comme par exemple une démarche d'ouverture de compte bancaire. <span style="background-color: #ffdddaff; padding: 2px; border-radius: 4px;">❌&nbsp;Ce cas d'usage ne peut pas être référencé sur Simplifions.data</span></li>
                <li class="fr-text--lg"><span style="background-color: #bfd9e3ff; padding: 2px; border-radius: 4px;"><b>OUI</b></span> <span style="background-color: #e7e7e7ff; padding: 2px; border-radius: 4px;">⬇️ Question suivante</span> </li>
            </ul>
        </li>
        <li>
            <p class="fr-pt-4w"><span class="fr-text--lg fr-text--bold">Ce cas d'usage peut-il être simplifié via l'utilisation de données administratives des usagers finaux* ? <br></span><i>* Particulier, entreprises ou associations</i></p>
            <ul>
                <li class="fr-text--lg"><span style="background-color: #bfd9e3ff; padding: 2px; border-radius: 4px;"><b>NON</b></span> <span style="background-color: #ffdddaff; padding: 2px; border-radius: 4px;">❌ Ce cas d'usage ne peut pas être référencé sur Simplifions.data</span></li>
                <li class="fr-text--lg"><span style="background-color: #bfd9e3ff; padding: 2px; border-radius: 4px;"><b>OUI</b></span> <span style="background-color: #e7e7e7ff; padding: 2px; border-radius: 4px;">⬇️ Question suivante</span> </li>
            </ul>
        </li>
        <li>
            <p class="fr-pt-4w"><span  class="fr-text--lg fr-text--bold">Ce cas d'usage concerne-t'il des acteurs publics différents ? <br></span></p>
            <ul>
                <li class="fr-text--lg"><span style="background-color: #bfd9e3ff; padding: 2px; border-radius: 4px;"><b>NON</b></span> Ce cas d'usage ne concerne qu'une seule administration car elle est la seule à en avoir les compétences, comme par exemple la DGFIP pour les déclarations d'impôts. <span style="background-color: #ffdddaff; padding: 2px; border-radius: 4px;">❌&nbsp;Ce cas d'usage n'a pas d'utilité à être référencé sur Simplifions.data</span></li>
                <li class="fr-text--lg"><span style="background-color: #bfd9e3ff; padding: 2px; border-radius: 4px;"><b>OUI</b></span> <span style="background-color: #e7e7e7ff; padding: 2px; border-radius: 4px;">⬇️ Question suivante</span> </li>
            </ul>
        </li>
        <li>
            <p class="fr-pt-4w"><span class="fr-text--lg fr-text--bold">Est-ce que des solutions, jeux de données ou API existent pour simplifier ce cas d'usage ? <br></span></p>
            <ul>
                <li class="fr-text--lg"><span style="background-color: #bfd9e3ff; padding: 2px; border-radius: 4px;"><b>OUI</b></span> <span style="background-color: #c3ffc5ff; padding: 2px; border-radius: 4px;">✅ Ce cas d'usage peut rejoindre le catalogue sur Simplifions.data</span>
                </li>
                <li class="fr-text--lg"><span style="background-color: #bfd9e3ff; padding: 2px; border-radius: 4px;"><b>NON</b></span> Même si aucune solution ne semble disponible, est-il possible de cartographier les données utiles pour simplifier ce cas d'usage ?</span>
                    <ul class="fr-ml-8w">
                        <li ><span style="background-color: #bfd9e3ff; padding: 2px; border-radius: 4px;"><b>NON</b></span> <span style="background-color: #ffdddaff; padding: 2px; border-radius: 4px;">❌ Ce cas d'usage ne peut pas être référencé sur Simplifions.data</span></li>
                        <li><span style="background-color: #bfd9e3ff; padding: 2px; border-radius: 4px;"><b>OUI</b></span> <span style="background-color: #c3ffc5ff; padding: 2px; border-radius: 4px;">✅ Ce cas d'usage peut rejoindre le catalogue sur Simplifions.data</span> </li>
                    </ul>
                </li>
            </ul>
        </li>
    </ol>
</div>
